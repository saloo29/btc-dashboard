#BTC/USDT Live Dashboard

A real-time Bitcoin dashboard build with React, TypeScript and Tailwind CSS.

##Live Demo
https://btc-realtime-dashboard-eight.vercel.app/

##Stack
- React +  TypeScript + Vite
- Tailwind CSS
- Bybit WenSocket API
- TradingView Advanced Chart Widget
- react-sparklines

##Features
- Live BTC price via Bybit WebSocket
- 24H High, Low, Turnover, Volume
- 60s sparkline mini chart
- TradingView advanced chart
- Light/Dark mode toggle

##Run Locally
npm install
npm run dev
